### 潮華（しおはな）｜物販LP

- 概要：海産物・乾物を扱うブランド「潮華」を想定した物販LP
- 目的：商品の魅力訴求とオンライン購入への導線設計
- 想定ターゲット：品質やストーリー性を重視する個人顧客
- 使用ツール：Studio
- 制作ポイント：
  - 商品写真を主役にしたビジュアル構成
  - 世界観を統一したトーン設計
  - 物販LPとしての情報量と余白のバランス
- 公開URL：https://olive982117.studio.site
